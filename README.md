# Web Content Fetcher

Приложение для поиска, скачивания и оффлайн-чтения веб-контента.

## Возможности
* **Поиск:** Поиск ссылок по ключевым словам.
* **Real-time прогресс:** Отображение загрузки через WebSockets.
* **Оффлайн:** Сохранение данных в LocalStorage.

## Технологии
* **Backend:** Node.js, Express, Socket.IO, Axios.
* **Frontend:** Vanilla JS, HTML, CSS.

## Запуск
1. Установите зависимости: `npm install`
2. Запустите сервер: `npm start`
3. Откройте: `http://localhost:3000`
